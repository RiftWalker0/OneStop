# One Stop (v1.4.0)

Minimal, stable skeleton with:
- Three themes (Pure White / Pitch Black / Material You).
- Drawer menu on all screens with centered titles.
- QS tile to toggle ADB + Dev options (requires `WRITE_SECURE_SETTINGS`).
- Updates screen with GitHub open + download intent.

> Note: `WRITE_SECURE_SETTINGS` must be granted via ADB or aShell.
